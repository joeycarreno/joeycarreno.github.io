Practice Assignment Links

# Assignment What I Learned
1 Getting Started with HTML5 HTML5 document structure, semantic elements (<header>, <nav>, <main>, <footer>), hyperlinks, images with alt text, and linking an external style sheet
2 Getting Started with CSS CSS selectors (descendant, child, sibling, attribute), color models (rgb(), hsl(), hex codes), typography, box model, list styles, dynamic pseudo-classes, and pseudo-elements
3 Designing a Page Layout Fluid vs. fixed layouts, CSS Grid (fr units, grid-column), positioning (relative, absolute, fixed), z-index, float & clearfix, and overflow
4 Graphic Design with CSS Background images, linear-gradient() and radial-gradient(), transform, filter, text-shadow, box-shadow, border-image, and opacity
5 Designing for the Mobile Web Responsive design with @media queries, mobile-first breakpoints, flexbox (flex-direction, flex-wrap, order, align-self), and print styles (@page, widows, orphans)
6 Working with Tables and Columns Table structure with row groups (<thead>, <tbody>, <tfoot>), column groups (<colgroup>), colspan, rowspan, border-collapse, and CSS multi-column layout
7 Designing a Web Form Form controls (text, email, tel, radio, checkbox, range, date, number), <select> with <optgroup>, <datalist>, and validation pseudo-classes
8 Enhancing a Website with Multimedia Embedding <audio> and <video> with <source> fallbacks, <iframe> embeds, CSS transition timing functions, and @keyframes animations
9 Getting Started with JavaScript External scripts with defer, document.getElementById(), innerHTML, the Date object, named functions with parameters, setTimeout(), and setInterval()
10 Arrays, Loops, and Conditionals Array literals, push(), forEach(), filter(), map(), for / while / do–while loops, comparison & logical operators, and if / else if / else statements

HTML & CSS Quick Reference

Type Tag / Property Category What It Does
HTML <header>, <footer> Page Structure Marks the top and bottom sections of a webpage
HTML <nav> Page Structure Wraps a block of navigation links into a menu
HTML <main>, <section>, <article>, <aside> Page Structure Semantic elements that describe the role of page content
HTML <h1> – <h6>, <p> Text Defines headings and paragraphs of text for hierarchy
HTML <a href=""> Links Creates a hypertext link; supports http, mailto, and tel URLs
HTML <img src="" alt=""> Media Embeds an inline image; alt text is required for accessibility requirements
HTML <table>, <thead>, <tbody>, <tfoot> Tables Structure tabular data with header, body, and footer row groups
HTML <form>, <input>, <select>, <textarea> Forms Collect user input through interactive form controls
HTML <audio>, <video> Media Embeds native audio and video players directly in the page
CSS color, background-color Color Set text and background colors using rgb(), hsl(), or hex values
CSS font-family, font-size, font-weight Typography Control typeface, size, and boldness of text
CSS margin, padding, border Box Model Define space outside, inside, and around an element
CSS display: flex, display: grid Layout Enable flexible and grid-based page layouts
CSS :hover, :focus, :nth-child(), :valid Pseudo-classes Applies styles based on user interaction or element position
CSS ::before, ::after, ::first-letter Pseudo-elements Inserts or styles content that is not present in the HTML source
CSS transform, filter, transition Effects Applies visual effects and smooth animated state changes
CSS @media, min-width, max-width Responsive Targets specific screen sizes for responsive and print design
